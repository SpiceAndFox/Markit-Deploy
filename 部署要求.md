# MarkIt A100 部署要求

## 选择

- 后端：`qwen2_5_vl`
- 主模型：`Qwen/Qwen2.5-VL-7B-Instruct`
- 推荐数据集：`Charades-STA`，比 ActivityNet 小，适合先跑 Moment Retrieval。

## 必备资源

```text
models/Qwen2.5-VL-7B-Instruct/      # Qwen2.5-VL 权重
data/charades/videos/               # 原始视频
data/charades/test.json             # 转成本仓库格式的测试 JSON
data/markit_overlay_videos/          # 完整 MarkIt 需要的 overlay 视频
outputs/                             # 输出目录
```

`test.json` 最小格式：

```json
[
  {
    "id": "video_001_1",
    "video": "video_001.mp4",
    "query": "a person opens a door",
    "start_time": 12.0,
    "end_time": 18.0,
    "duration": 60.0,
    "nouns": {"1": "person", "2": "door"}
  }
]
```

overlay 命名必须和 `id` 对齐：

```text
data/markit_overlay_videos/<id>_overlay.mp4
```

## 完整 MarkIt 额外模型

按论文配置还需要：

```text
Qwen-7B       # subject noun extraction, Kmax=3, temperature=0
YOLOE-Large   # text-conditioned instance mask grounding
```

本仓库不包含完整的 YOLOE mask/overlay 生成脚本，需要你单独补齐并提前生成 `<id>_overlay.mp4`。没有 overlay 时脚本会退化为原始视频加帧号，不是完整 MarkIt。

论文默认渲染配置：

```text
mask alpha=0.3, contour beta=1.0, contour width=3
frame index: bottom_right, font_size=38, color=black
```

## Docker Compose

镜像只放运行环境；代码、模型、数据、HF 缓存、输出都通过 volume 挂载。

服务器上：

```bash
cp .env.example .env
```

编辑 `.env`：

```text
MARKIT_MODEL_ROOT=/srv/markit/models
HF_CACHE_DIR=/srv/markit/hf-cache
MARKIT_DATA_ROOT=/srv/markit/data
MARKIT_OUTPUT_ROOT=/srv/markit/outputs
QWEN25_VL_MODEL_PATH=/models/local/Qwen2.5-VL-7B-Instruct
OVERLAY_ROOT=/data/markit_overlay_videos
RAW_VIDEO_ROOT=/data/charades/videos
TEST_PATH=/data/charades/test.json
```

如宿主机需要代理访问 GitHub/Hugging Face/PyPI，在 `.env` 里加：

```text
DOCKER_HTTP_PROXY=http://host.docker.internal:7890
DOCKER_HTTPS_PROXY=http://host.docker.internal:7890
DOCKER_ALL_PROXY=http://host.docker.internal:7890
DOCKER_NO_PROXY=localhost,127.0.0.1,host.docker.internal
```

端口按你的本地代理实际端口改。不要写 `127.0.0.1`，容器里它指向容器自己。建议填 HTTP 代理地址，`pip install` 更稳。

如果 Docker 在远端 A100 服务器上运行，`host.docker.internal` 指的是服务器本机；需要在服务器上有代理，或先把你本地代理通过 SSH 隧道转发到服务器。

构建和运行：

```bash
docker compose build
docker compose up markit
```

模型下载不要放进 Docker 流程，先在宿主机或服务器上直接跑：

```bash
python -m pip install -U huggingface_hub hf_transfer hf_xet
python tools/download_models.py --model-root /srv/markit/models
```

如果宿主机下载也需要代理，用宿主机可访问的地址：

```text
DOWNLOAD_HTTP_PROXY=http://127.0.0.1:7890
DOWNLOAD_HTTPS_PROXY=http://127.0.0.1:7890
DOWNLOAD_ALL_PROXY=http://127.0.0.1:7890
```

调试：

```bash
docker compose run --rm markit bash
```

默认不启用 FlashAttention。基础流程跑通后如需尝试，在 `.env` 中设：

```text
INSTALL_FLASH_ATTN=1
```

## 单机命令参考

```bash
python eval/vlm_mr_markit.py \
  --backend qwen2_5_vl \
  --model_path /path/to/Qwen2.5-VL-7B-Instruct \
  --overlay_root /path/to/markit_overlay_videos \
  --raw_video_root /path/to/charades/videos \
  --test_path /path/to/charades/test.json \
  --save_path outputs/predictions/charades_markit_qwen25vl.json \
  --max_samples -1
```
